Welcome to the open source browser game `TheShareGame <https://www.thesharegame.com/>`_.

The source code can be found on `Github <https://github.com/Zarathustra2/TheShareGame>`_.

This game has been built with the following technologies:

- `Django <https://www.djangoproject.com/>`_ for the Api
- `Golang <https://golang.org/>`_ for the web sockets
- `Vue <https://vuejs.org/>`_ for the frontend
